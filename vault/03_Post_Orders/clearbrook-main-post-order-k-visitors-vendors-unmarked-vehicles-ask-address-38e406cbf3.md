# Clearbrook Main Post Order - Visitors Vendors Unmarked Vehicles Ask Address

## Rule

For visitors and vendors in unmarked vehicles, ask for the address and res name. If access is not found, contact the res once, allowing up to five rings only. Do not leave a voicemail. If there is no response, deny entry.
