# For Sierra Ridge, All Overnight Visitors Must Present Physical Id Before Access Is Granted

## Summary

All overnight visitors to Sierra Ridge must present a physical ID before being granted access.

## Details

This policy applies to all overnight visitors at Sierra Ridge. The requirement is to present a physical form of identification prior to access being granted.

## Agent Action

Ensure that all overnight visitors are informed of and comply with the physical ID requirement before granting access.
