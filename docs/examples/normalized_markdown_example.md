# Post Order

## Summary
Physical IDs are mandatory for all visitors.

## Exact Rule
Digital IDs are not accepted.

## Applies To
All visitors and vendors.

## Required Verification
Physical government-issued ID.

## Agent Action
Request physical ID before granting access.

## Deny / Approve Logic
Deny access if no physical ID is provided.

## Required Script
"Unfortunately digital IDs are not accepted for this community."

## Escalation Rules
Escalate if visitor becomes confrontational.

## Exceptions
None.

## QA Risk
Immediate QA failure if digital ID is accepted.
